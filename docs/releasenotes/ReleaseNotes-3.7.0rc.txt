Release Notes
xTuple Desktop (xtDesktop)
Version 3.7.0rc
March 11, 2011
==================================

This is the release candidate of the 3.7.0 xTuple Desktop package,
a free and open source extension package for xTuple ERP (PostBooks, 
Standard, and Manufacturing Editions). 

xtDesktop is available under the same Common Public Attribution 
License (CPAL) as the PostBooks Edition of xTuple ERP 
(www.xtuple.com/CPAL). For more information on xTuple's extension 
package architecture, please see www.xtuple.org/developer-zone.

==================================

Bug fixes in xtdesktop 3.7.0rc:

* Sales Activity Screen no showing correctly
* 'Open' option in the Manufacture History dock is unresponsive
* POs for Non-Inventory Items do not appear on the Desktop Open PO list


Release Notes
xTuple Desktop (xtDesktop)
Version 3.7.0alpha
January 31, 2011
==================================

This is the alpha release of the 3.7.0 xTuple Desktop package,
a free and open source extension package for xTuple ERP (PostBooks, 
Standard, and Manufacturing Editions). 

xtDesktop is available under the same Common Public Attribution 
License (CPAL) as the PostBooks Edition of xTuple ERP 
(www.xtuple.com/CPAL). For more information on xTuple's extension 
package architecture, please see www.xtuple.org/developer-zone.

==================================

Changes to the desktop in this release are exclusively to support 
core changes in xTuple 3.7.0.  Core changes made that affect the 
desktop are listed below. Additional detail for each item listed below
may be found on our community website (www.xtuple.org).

Support for New Features in 3.7:

* [Manufacture] Consolidate Work Order Schedule Reports
* [Inventory] Consolidate Inventory Availability Reports
* [Sales] Consolidate Sales History Reports
* [Sales] Consolidate Bookings Reports
* [System] License Manager enhancement/policy changes for 3.7
